# Contributing to Prompt Forge

Thanks for your interest in improving Prompt Forge. This is a community project and contributions are welcome.

## How to contribute

1. **Fork** the repository
2. **Create a branch** for your change
3. **Make your changes** and test them by running the skill in Claude Code
4. **Submit a pull request** with a clear description of what you changed and why

## What to contribute

**Improving existing phases** — If you've found that a phase consistently misses a certain class of failure, propose a fix. Include before/after examples showing the improvement.

**Domain-specific adaptations** — The pipeline is general-purpose. If you've adapted it for a specific domain (coding prompts, creative writing, data analysis, etc.) and it works well, consider contributing it as a variant.

**Documentation and examples** — Real-world examples of prompts built with the pipeline help new users understand what to expect.

**Bug reports** — If a phase produces unexpected results, open an issue describing the input, what you expected, and what happened instead.

## Guidelines

- Keep the pipeline lean. Every phase and instruction must earn its place.
- If you propose a new phase, explain what category of failure it catches that existing phases miss.
- Test your changes by actually running the skill in Claude Code.
- Write clear commit messages that explain the "why", not just the "what".

## Code of conduct

Be kind, be constructive, assume good intent. This is a small project — let's keep it welcoming.
